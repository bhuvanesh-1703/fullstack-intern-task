# Mini SaaS Template Store

A simple full stack web application built using React.js, Node.js, Express.js, and MongoDB.  
Users can register, login, view templates, add favorites, and manage their favorite templates.

---

# Tech Stack Used

## Frontend
- React.js
- TailwindCSS
- Axios
- React Router DOM
- Sweetalert

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt
- Nodemailer

---

# Features

- User Registration & Login
- JWT Authentication
- Password Hashing
- View Templates
- Add to Favorites
- Favorites Page
- Responsive UI
- Protected Routes
- Welcome Email using Nodemailer
